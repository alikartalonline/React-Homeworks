### JSON Server

```
https://github.com/typicode/json-server
```

<br/>

### Install JSON Server

```
npm i json-server
```

<br/>

### Start JSON Server 
##### (Check > Package.json/Script)

```
json-server --watch employee.json  (or your example.json)

``` 

### Start Server

```
npm run start:server 
```
